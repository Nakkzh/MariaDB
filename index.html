Comandos DDL (Data Definition Language)

CREATE DATABASE
Cria um novo banco de dados no sistema.

USE
Define qual banco de dados será utilizado nas próximas operações.

CREATE TABLE
Cria uma nova tabela com colunas e tipos de dados definidos.

ALTER TABLE
Modifica a estrutura de uma tabela existente, como adicionar ou remover colunas.

DROP COLUMN
Exclui uma coluna específica de uma tabela.

DESCRIBE / DESC
Exibe a estrutura de uma tabela, incluindo nomes de colunas, tipos e restrições.

Comandos DML (Data Manipulation Language)

INSERT INTO
Insere novos registros em uma tabela.

UPDATE
Altera os dados de registros existentes em uma tabela.

DELETE FROM
Remove registros de uma tabela com base em uma condição.

Cláusulas de Filtro

WHERE
Define uma condição para selecionar ou modificar registros.

AND
Combina múltiplas condições em uma cláusula WHERE.

OR
Retorna resultados que atendem a pelo menos uma das condições.

NOT
Inverte o resultado de uma condição, excluindo registros que a satisfazem.

LIKE
Realiza buscas por padrões em valores de texto.

Consultas e Relacionamentos

SELECT
Recupera dados de uma ou mais colunas de uma tabela.

JOIN
Relaciona dados entre duas ou mais tabelas com base em uma chave comum.
